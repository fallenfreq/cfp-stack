Just some rough examples

Welcome to the Interactive Editor!

This is a demo showcasing the flexibility of the editor. You can use registered Vue components, apply custom styles and classes, embed YouTube videos, display images, add tables, and even work with syntax-highlighted code blocks.


The editor allows you to switch between a WYSIWYG view and an editable code view. This means you can directly modify the content or structure as code and see the changes live. Note that the purpose here is to show the code, not necessarily render the final HTML output. Let’s explore some examples below:


Example 1: Embedded YouTube Video

This is an example of embedding a YouTube video using an iframe. You can add any video by simply pasting the iframe code inside the editor. The video will be embedded in the output, but here we are showing the code used to achieve this.

Example 4: JavaScript Code with Syntax Highlighting

The editor also supports syntax highlighting for code. Below is an example of a simple FizzBuzz JavaScript program. The editor automatically highlights the syntax based on the language specified, in this case, JavaScript. 


// Simple JavaScript FizzBuzz example
for (var i = 1; i <= 20; i++) {
  if (i % 15 === 0) {
    console.log("FizzBuzz");
  } else if (i % 3 === 0) {
    console.log("Fizz");
  } else if (i % 5 === 0) {
    console.log("Buzz");
  } else {
    console.log(i);
  }
}
